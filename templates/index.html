Welcome to Genesis....

Genesis is a platform for our local farmers to be able to gain insights on the soil on their farms and how well to utilise that soil. This is done by giving farmers the best recommendation on which crop can thrive in their soils using :

Internet of Things

Machine Learning

Web Development

GENESIS, A SMART WAY TO FARM!!

Genesis is Web Based application that has been integrated with a machine learning model which recommends an appropriate crop to be planted with reference to different soil properties. Using IoT devices, soil monitoring has been made possible. This is accomplished by collecting and analyzing soil data. The web application takes input from farmers and submits that data to the machine learning model which then analyzes the data then gives a recommendation to the farmer. Input collected includes:

  • Nitrogen Levels [N]
  • Phosphorate Levels [P]
  • Potassium Levels [K]
  • Temperature
  • pH
  • Rainfall

IoT Devices

Genesis uses IoT devices to collect the soil properties stated above. A diagram representation of these devices is shown on the Left. The metal stips attached to it collect these data and display real time information on the display screen embedded to them.

Machine Learning

For the prediction to be done, we need a dependable model that can give us accurate results. for these results to be obtained, Genesis used Logistic regression as a machine learning model to accomplish its objectives. The model collects soil data from the IoT devices, then using a trained model, it predicts which crop could flourish given the soil properties given. Without machine learning, the prediction can not happen.

Web Application

The model so far is partially complete. or it to be fully complete and functional, we need a link thet will join the Machine Learning model to the data collected by the IoT devices. The web application provides a user-friendly user interface that explains the services offered, an explanation of the project and most importantly, a platform where users can put in soil properties to be analysed and for a prediction to come from the data. Genesis combines the web application to the machine learning model as it is easier for farmers to understand and use. Through our services section, farmers can easily input the collected soil properties for the model to predict.

Conclusion

A combination of the components stated above make up GENESIS!! which provides a smart way to farm.
